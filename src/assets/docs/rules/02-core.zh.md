# 二、基础规则（卡组、区域与胜负）

## 卡组构成（常规决斗）

以下为「官方规则书」与大众对战常用的基准；**官方大赛**可能在副卡组人数、副卡组交换流程等方面另有细则，参赛前请读当场「大赛手册」。

| 项目 | 张数 / 说明 |
|------|----------------|
| 主卡组 | **40～60** 张 |
| 额外卡组 | **0～15** 张（融合·同调·超量·连接等从中出场的怪兽） |
| 副卡组 | **0～15** 张；在三局两胜之间可与主卡组、额外卡组做等量置换 |

同名卡在主卡组、额外卡组、副卡组中合计通常不超过 **3** 张（禁卡表指定为 0/1/2 张的除外）。

## 主要区域说明

理解区域名称，才能读懂效果里的「场上」「墓地」「除外」「魔法与陷阱区域」等用语。

| 区域 | 要点 |
|------|------|
| 卡组 | 面朝下；通常从上方抽牌，除非效果指定 |
| 手牌 | 仅持有者可见；张数一般无上限（个别比赛可能有调度细则） |
| 墓地 | 被破坏、解放、送墓的效果终点等（具体依效果用语） |
| 除外 | **游戏外**区域，与墓地不同 |
| 场地区域 | **场地魔法**专用；新场地通常会顶掉旧场地 |
| 魔法与陷阱区 | 合计通常 **5** 格；永续、装备等持续在场上的卡占格 |
| 怪兽区 | **主要怪兽区（MMZ）** 一般 **5** 格；另有 **额外怪兽区（EMZ）** 用于连接怪兽及摆放规则（见「大师规则要点」） |
| 灵摆区 | 放置灵摆刻度相关卡片的位置（摆放规则随大师规则更新） |

## 表示形式（怪兽）

- **表侧攻击**：正面朝上，攻击表示。
- **表侧守备**：正面朝上，守备表示。
- **里侧守备**：背面朝上盖放；被攻击时翻开。

连接怪兽没有守备表示；部分情况下不能变更表示形式，依卡面与规则。

## 胜负与平局（常见）

| 结果 | 常见原因 |
|------|-----------|
| 一方获胜 | 对方 LP 为 0；对方认输；达成合法的特殊胜利 |
| 一方败北 | 自己在必须抽卡时卡组为空；LP 为 0；认输 |
| 平局（少见） | 同时达成胜负条件、或规则认定的双败等（大赛可有额外规定） |

## 决斗的基本礼仪（实用）

- 宣言阶段与步骤清晰：「进入战斗阶段」「攻击宣言」等。
- 连锁结算顺序：**后发先至**（先发动的后结算）。
- 有疑问优先沟通；竞技环境以裁判解释为准。

## 决斗开始前（建议流程）

1. **问候与确认**：是否与陌生人首次对战、是否使用最新禁卡表与 SPY 规则（若有）。
2. **卡组公示**：正规赛需提交卡组表；友谊赛可口头确认是否允许副卡组、是否使用同一 SPY。
3. **随机决定先后攻**，再各自洗牌；勿在对方缺席时独自掷骰「代掷」。

## 洗牌、切牌与「不当洗牌」

- **洗牌**目标：顺序不可预测；避免刻意保留特定排列。
- **切牌**：对方可在洗牌后切牌或拒绝（依比赛规定）；休闲对战通常允许对方切一次。
- **掉卡、亮牌**：意外掉落或翻开可视情况重洗相关部分，**以双方同意或裁判指示为准**。

## 控制权与「谁的卡」

- **控制者**：怪兽与多数永续魔陷的「控制者」决定能否发动其起动效果等。
- **所有者**：卡归属哪一方的卡组/额外，不因暂时夺取控制权而改变「原本持有者」在部分效果中的判定——**以数据库用语为准**（与「控制」「持有」「场上」等区分）。

## 计数与公开信息

- **LP**：可用纸笔、骰子或电子设备记录；变更时应双方确认数值。
- **公开情报**：墓地、除外、表侧场上卡名与攻守等一般视为公开；**卡组内顺序**通常非公开，除非效果翻开检索。
