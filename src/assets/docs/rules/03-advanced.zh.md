# 三、对局流程、召唤与战斗（进阶）

## 回合结构（再细化）

典型回合包含：**抽卡 → 准备 → 主要阶段 1 →（战斗）→ 主要阶段 2 → 结束**。  
是否进入战斗阶段、何时优先权转移，由玩家选择与规则共同决定。

### 抽卡阶段

- 通常抽 **1** 张。
- 「抽卡阶段发动」的效果时点在此阶段（具体能否插入连锁依咒文速度与规则）。

### 准备阶段

- 少数持续效果或诱发在此阶段检查。

### 主要阶段

- **通常召唤／盖放**（受每回合通召次数限制，除非效果增加）。
- 发动魔法、陷阱（满足速度与条件）、起动怪兽效果等。
- **特殊召唤**在不违反召唤手续的前提下进行。

### 战斗阶段

- 只有表侧攻击表示的怪兽可以宣言攻击（另有「可直接攻击」等例外）。
- **攻击宣言 → 伤害步骤前后** 有不同的诱发与反击窗口（详细裁定以数据库与裁判为准）。
- 守备表示怪兽被攻击：比较攻击方的攻击力与守备方的守备力决定是否破坏、是否造成贯通伤害等。

### 结束阶段

- 手牌数量超过上限时需要舍弃（规则规定的上限，通常为 **6** 张，依当时规则）。
- 「回合结束时」「结束阶段时」的效果。

## 召唤类型纲要

### 通常召唤与上级召唤

- 每回合通常召唤权 **1** 次（除非效果写明「追加通召」等）。
- **上级召唤**以解放场上怪兽为代价，召唤更高星级的怪兽。

### 特殊召唤（概述）

| 类型 | 记忆要点 |
|------|-----------|
| 融合 | 依「融合」卡或规则，素材送墓/除外等后从额外卡组融合召唤 |
| 同调 | 调整 + 非调整，星级合计匹配 |
| 超量 | 若干只**等级相同**的怪兽叠放为素材 |
| 连接 | 素材满足连接怪兽要求，从额外卡组连接召唤；有 **LINK** 值与箭头 |
| 仪式 | 仪式魔法 / 仪式怪兽手续满足后仪式召唤 |
| 灵摆 | 依刻度与规则从手牌·额外卡组灵摆召唤 |

具体解放几只、是否取对象、是否被无效，一律以**卡面**与**最新数据库裁定**为准。

## 战斗伤害（概念）

- **战斗伤害**：战斗步骤中因战力比较对 LP 造成的伤害。
- **效果伤害**：卡的效果文本造成的伤害（与战斗伤害区分，部分卡仅针对其中一种）。

## 何时能「发动」与「连锁」

- 同一时点多个效果欲发动时，按规则组成**连锁**依次结算。
- **咒文速度**概念用于判断某些效果能否连锁另一些效果（例如反击陷阱多为速度 3）。
- 「起动效果」「诱发效果」「永续效果」等在裁定上差异很大，对战中有争议应查询官方数据库或请教裁判。

## 常用关键词（遇到再查数据库）

**解放、破坏、除外、送入墓地、控制权、取对象／不取对象、战斗破坏、抗性（不受…影响）** 等，均以卡面记述与最新 FAQ 为准；本百科只做指路，不替代数据库。

## 优先权与「能不能在这步做事」

- **回合玩家**在自己回合多数阶段拥有先提议发动、先进入下一阶段的「气氛上」优先，但**具体能否发动**仍取决于：时点、咒文速度、是否已有连锁正在结算中等。
- **对方回合**想发动速攻魔法或部分陷阱时，必须满足对应时机；若不确定，应 **暂停确认** 而非默认「可以」。

## 主要阶段 2 与「战斗阶段跳过」

- 若**未进入战斗阶段**，可从主要阶段 1 直接进入结束阶段（依规则许可）。
- 若进入了战斗阶段，之后仍有 **主要阶段 2**，用于战斗后再展开或盖伏。
- **速攻魔法**在对方回合在自己场上发动等细则，以 Advanced Rules 与单卡 FAQ 为准。

## 伤判里「少做事」的原因（概念）

伤害步骤被分成多个极细的「小步骤」，许多效果被规则禁止在特定小步骤发动，是为了避免结算顺序爆炸性复杂。**实战不确定时**：先查 Advanced Rules 图表，再查涉及卡片的 FAQ。

## 里侧与「查看」

- **里侧除外、里侧额外**等表述与能否发动某些效果密切相关。
- **翻开确认**与「查看卡组最上方」不同；是否构成「公开情报」影响诱发——**以数据库对每张卡的说明为准**。
